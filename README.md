# DBMS_LAB
DBMS lab assignment

This repository contains my Database Management Systems (DBMS) assignment, created as part of the academic curriculum. The objective of this assignment is to understand and implement core DBMS concepts through structured queries, database design, and practical examples.

## Technologies Used

Database: Oracle <br>
Language: SQL <br>
Tools: SQL*Plus / Oracle SQL Developer <br>
Platform: GitHub

## How to Run SQL Scripts in Oracle

Open Command Prompt / Terminal
Login to Oracle:
sqlplus username/password
Navigate to the directory containing SQL files:
cd path_to_sql_scripts
Execute the script:
@script_name.sql
The output will be displayed directly in the terminal
